# SCU-SelectCourseScript

本脚本为简化四川大学选课而生.

## 用户须知:

- 本脚本只供测试使用,**不得使用于实际选课阶段**.
- 请下载本软件**24小时后自行删除**.
- **不得随意传播**
- 用户的操作一概**与本人无关**,本人**不负**相关连带责任.
- 脚本相关解释权归本人所有

## 使用:

- 请在电脑上安装`python3`运行环境,并安装以下库:
  - requests
  - BeautifulSoup
  - Image

- 请用命令提示符/Bash切换到本软件目录下,运行'python selectCourse.py',根据提示执行即可.
